# goodweather-ios
Weather App that consumes the [OpenWeather API](https://openweathermap.org/).


## Run Application
1. Open terminal 
2. Clone this repository\
   `git clone https://github.com/marianobattaglia/goodweather-ios.git`
3. Open project file GoodWeather.xcodeproj
